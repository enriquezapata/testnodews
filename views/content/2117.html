9.30.3.2. Strip Direction and End Joints

(1) wood strip flooring="http://thehandyforce.com/flooring/" title ="Handyman flooring install Toronto">flooring shall not be laid parallel to lumber subflooring="http://thehandyforce.com/flooring/" title ="Handyman flooring install Toronto">flooring unless a separate underlay is provided.

(2) If wood strip flooring="http://thehandyforce.com/flooring/" title ="Handyman flooring install Toronto">flooring is applied without a subfloor, it shall be laid at right angles to the joists so that the end joints are staggered and occur over supports or are end matched.

(3) If the flooring="http://thehandyforce.com/flooring/" title ="Handyman flooring install Toronto">flooring is end matched, it shall be laid so that no two adjoining strips break joints in the same space between supports and each strip bears on no fewer than two supports.
